## Sobre a Callix

A [Callix](https://callix.com.br) é uma plataforma brasileira de **contact center na nuvem** que unifica voz, WhatsApp e SMS para operações de atendimento, vendas e cobrança.

Para a visão completa da organização, veja o [README do perfil](profile/README.md).
